## ✈️ 프로젝트 소개

“여행을 조금 더 편하고, 더 가치 있게 만들 수는 없을까?”

<aside>
*“여행을 조금 더 **편하고**, 더 **가치** 있게 만들 수는 없을까요?”*

프로젝트 **모행**은 이 작은 질문에서 시작되었습니다.
모행에서는 여행을 혼자 계획하는 부담을 덜고,
다른 사람들과 여행 경험을 공유하며 함께 즐길 수 있어요.

몇 가지 간단한 질문에 답하고 여행 계획을 알려주세요.
AI가 당신의 취향에 꼭 맞는 **맞춤형 여행 일정**을 만들어드립니다.

또한 다른 사람들의 여행 일정과 여행 일지를 둘러보고 소통할 수 있어요.
마음에 드는 일정을 수정하거나 참고해 **나만의 여행 계획**으로 완성해보세요.

모행과 함께라면, 
여행 준비는 **가볍게** — 여행의 순간에 더욱 **집중**할 수 있을 거예요 ✨

</aside>
